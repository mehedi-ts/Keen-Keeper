## 📌 Project Overview

Keen Keeper is a modern friend management web application designed to help users maintain meaningful relationships. Users can view friends, explore detailed profiles, and interact through Call, Message, and Video Call features. Each interaction is tracked and stored in a timeline, with a visual statistics dashboard for better insights.

---

## 🚀 Features

* 📋 Dynamic friend list with detailed profile pages  
* 📞 Call, 💬 Message, 🎥 Video Call actions  
* 🔔 Real-time toast notifications for interactions  
* 🕒 Timeline tracking system for all activities  
* 📊 Statistics dashboard with pie chart visualization  
* 📱 Fully responsive (Mobile, Tablet, Desktop)  

---

## 🛠️ Tech Stack

* ⚛️ React.js  
* 📜 JavaScript (ES6+)  
* 🎨 Tailwind CSS + DaisyUI  
* 🔀 React Router DOM  
* 📊 Recharts  
* 🔔 React Hot Toast / React Toastify

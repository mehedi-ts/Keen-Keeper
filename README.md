# 👥 Keen Keeper

## 📌 Project Description
Keen Keeper is a friend management web application where users can view friends, open detailed profiles, and interact through Call, Message, and Video Call actions. Each interaction shows a toast notification and is saved in a timeline. The app also includes a statistics page with a pie chart to visualize user activity.

---

## ⚙️ Technologies Used
- React.js
- JavaScript (ES6+)
- Tailwind CSS
- DaisyUI
- React Router DOM
- Recharts
- React Hot Toast / React Toastify

---

## ✨ Key Features

- Dynamic friend list with detail page navigation  
- Call, Message, Video Call actions with toast notifications  
- Timeline system to track all interactions  
- Statistics page with pie chart visualization  
- Fully responsive design (mobile, tablet, desktop)  

---


## 📱 Responsiveness
The application is fully responsive and works smoothly on:
- Mobile devices  
- Tablets  
- Desktop screens
